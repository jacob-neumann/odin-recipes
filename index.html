Background

This site is a project done as a part of The Odin Project's Foundations course, allowing me to practice with html and git basics. My ultimate goal is to use web design to create web-accessible tools for work and personal use. Web applications are ideally suited for many of my use cases, as it is difficult to get permission to write and execute code on my work computers. While this project does not involve programming, it is necessary to learn how to make web applications usable (I don't plan on running a command line within a browser to interface with my backend software).
